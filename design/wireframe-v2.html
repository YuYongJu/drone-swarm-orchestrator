Ground Station Wireframes v2 — Mission Feed Concept

v2 — FEED-FIRST DESIGN
Design concept: The Mission Feed is a persistent, real-time scrolling log — like a livestream chat crossed with a terminal. On desktop, it lives as a right panel alongside the map. On mobile, it becomes the primary interface. Critical alerts streak across the map as overlay banners. Operators read the mission like a radio log.

Desktop: 3-Column Layout (Fleet | Map | Feed)

The main view. Fleet sidebar left, satellite map center, Mission Feed right. Alert overlays streak across the map for critical events. Operator can type commands directly into the feed.
